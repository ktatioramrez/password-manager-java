# Gestor de Contraseñas Seguro - Java Web
Proyecto final para el taller de programación en Java.

### Tecnologías usadas:
* **Backend:** Java Servlets, Maven.
* **Base de Datos:** MySQL.
* **Seguridad:** * Cifrado de credenciales: **AES-256-GCM**.
  * Hash de usuarios: **BCrypt**.

### Actividad con fines educativos.
